# Hotel Booking Database Project
  
This project demonstrates the design and implementation of a **hotel booking system** using **MySQL** and **PHP**.

---

## Project Overview

The system allows users to view, manage, and query hotel booking information through a PHP interface connected to a MySQL database.

It includes:
- SQL database schema (`sarya_demir.sql`)
- PHP scripts for interacting with the database (`part1get.php`, `SelectHotel.php`, etc.)
- Sample data in CSV format for testing

---

## Features
- View and search hotels, rooms, and travel agents  
- Manage bookings and facilities  
- Retrieve data from structured CSV files  
- Relational database design demonstrating entity relationships:
  - **Hotel**, **Room**, **RoomType**, **Booking**, **Facility**, **TravelAgent**, **City**, **District**

---

## Technologies
- PHP  
- MySQL  
- HTML  
- CSV data import

---

## Files
- `sarya_demir.sql` — Database schema and sample data
